# 🌐 Personal Blog

这是一个基于开源项目搭建的个人博客，用于记录学习笔记、技术总结和日常思考。

## ✨ Features
- 📝 支持 Markdown 写作
- ⚡ 快速构建与部署
- 🎨 简洁美观的页面风格
- 📂 文章分类与标签管理

## 🚀 Usage
```bash
# 安装依赖
pnpm install

# 本地运行
pnpm dev

# 构建项目
pnpm build
```
## 📌 Purpose

### 本博客主要用于：

- 记录 AI / 机器人 / 编程学习过程
- 整理技术知识与项目经验
- 构建个人技术成长轨迹
